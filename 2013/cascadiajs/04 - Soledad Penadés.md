**audio tags (web component)**
* `<audio-context>`
  * context for creating audio
* `<audio-mixer>`
  * mix child elements
* `<audio-oscillator>`
  * create an oscillator
* `<audio-chain>`
  * feed one audio source through another
* `<audio-oscilloscope>`
  * canvas display of oscillator
* `<audio-filter>`
  * filter input in a chain
